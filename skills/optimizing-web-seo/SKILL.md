# SEO para páginas públicas

## Alcance

La aplicación administrativa autenticada no necesita posicionamiento de contenido interno. Esta skill se usa para una futura página pública, catálogo o landing de MoorePrint.

## Principio

Crear páginas útiles para una intención real de búsqueda. No duplicar contenido ni llenar textos con palabras clave.

## Requisitos

- Un título único y descriptivo por página.
- Meta descripción específica.
- Un encabezado principal coherente.
- URL legible.
- Canonical correcta.
- Open Graph y vista previa social.
- Imagen compartible optimizada.
- Contenido indexable sin depender únicamente de JavaScript.
- Enlaces internos claros.
- Sitemap y robots revisados.
- Datos estructurados cuando correspondan.

## SEO local de MoorePrint

Mantener consistencia en:

- nombre del negocio;
- servicios;
- zona atendida;
- horario confirmado;
- WhatsApp `722 503 8566`;
- redes y dirección confirmadas.

No inventar ubicación, horario, precios ni testimonios.

## Páginas de servicio

Una página de servicio debe responder:

- qué producto se ofrece;
- materiales o acabados;
- tamaños y personalización;
- tiempo o proceso, solo si está confirmado;
- cómo solicitar cotización;
- fotografías reales o representativas autorizadas;
- preguntas frecuentes específicas.

## Calidad técnica

- Imágenes con tamaño y texto alternativo adecuados.
- Buen rendimiento móvil.
- Sin enlaces rotos.
- Estado HTTP correcto.
- No indexar paneles privados, pruebas o entornos temporales.

## Errores comunes

- Intentar posicionar la aplicación administrativa.
- Crear muchas páginas casi idénticas por ciudad o producto.
- Usar textos genéricos generados sin información del negocio.
- Colocar datos estructurados que no coinciden con el contenido visible.
